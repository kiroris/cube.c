# 3D cube in the matrix

![cube](cube.jpg)

### Building:
```
$ git clone https://github.com/kiroris/cube.c.git
$ cd cube.c 
$ gcc cube.c -o cube -lm -O3   # gcc compiler
$ clang cube.c -o cube -lm -O3 # clang compiler
```
